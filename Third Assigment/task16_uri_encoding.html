URI Encoding Examples

This page demonstrates proper URL encoding for special characters

URL with Special Characters

Incorrect URL (with special characters):

https://example.com/search?query=HTML & CSS

Correct URL (properly encoded):

https://example.com/search?query=HTML%20%26%20CSS

URI Encoding Reference

Character Encoded Form Description
Space %20 Most common encoding needed
& %26 Ampersand character
+ %2B Plus sign
= %3D Equals sign
? %3F Question mark
# %23 Hash/fragment identifier

More URL Examples

  • Search for "web development":
    https://example.com/search?q=web%20development
  • Search with special characters:
    https://example.com/search?q=HTML%2BCSS%26JavaScript
  • Path with spaces:
    https://example.com/my%20documents/file%20name.html
